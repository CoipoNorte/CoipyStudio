## Features

- **Python 3.11** via Pyodide (WebAssembly)
- **pip install** — install packages from PyPI
- **File explorer** — create, rename, duplicate, delete files & folders
- **Tabs** — multi-file editing
- **Context menu** — right-click actions
- **3 Themes** — CoipyDark, Dracula, Light
- **3 Languages** — English, Español, Português
- **Pyflakes linter** — syntax checking before run
- **Matplotlib** — plots rendered in-browser
- **Upload files** — drag & drop support
- **Download project** — export all files
- **Resizable panels** — sidebar, editor, terminal
- **Terminal layout** — toggle right/bottom with one click

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Tech

React 18 + Vite + Zustand + CodeMirror 5 + Pyodide
